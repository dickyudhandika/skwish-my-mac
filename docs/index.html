what it does

the useful parts, not the bloated theatre.

This page is grounded directly in the skwish-my-mac README, so the promise matches the app instead of marketing fan fiction.

monitor

Track live cpu, memory, disk, and network health from the menu bar. Then use Quick Clean for the obvious junk first.

  • live system stats
  • health score at a glance
  • quick clean for common caches

apps

Scan /Applications, estimate app size, check last-used date, and uninstall selected apps with a warning when sudo is needed.

  • estimated app sizes
  • last used visibility
  • selective uninstall flow

cleanup

Find high cpu processes, memory hogs, and disk junk. Clean only what you check. No mystery cleanup button bullshit.

  • kill high cpu processes
  • memory purge action
  • selective disk cleanup
how it works

three simple steps.

skwish-my-mac keeps the workflow boring on purpose. Boring is good when files are involved.

1

scan what matters

Read live health data, app usage clues, and common junk locations to see what is actually worth touching.

2

review before cleaning

Select the specific items you want to clean, uninstall, purge, or kill. The point is visibility, not magic.

3

clean safely

Many cleanup actions go to ~/.Trash first, which is a lot less terrifying than a giant hidden delete button.

safety

safer by default.

The README is clear about two things: several actions use trash-first cleanup, and root-owned app removal may still need sudo.

what makes it feel honest

  • trash-first cleanup where possible
  • selective cleanup, only checked items
  • sudo warning when elevated permissions are required
  • github release check instead of weird silent auto-installs

what it does not pretend to be

  • not a fake “your mac is dying” dashboard
  • not a black-box optimizer
  • not a background drama machine
  • not an auto-delete roulette wheel

clean your mac, sure. just don’t make the app itself feel like malware with better branding.

run locally

dead simple setup.

skwish-my-mac currently ships as a Swift macOS app, so the easiest CTA is just: clone it and run it.

Requirements: macOS 13+, Xcode Command Line Tools or full Xcode, Swift 5.7+.

cd ~/Documents/skwish-my-mac
swift build
swift run
